# 🧠 BHT QuizApp: Interactive Assessment Engine

An engaging, dynamic quiz application designed to test user knowledge with real-time feedback and performance metrics.

## ✨ Features
* **Dynamic Challenge Feed:** Pulls and presents multiple-choice questions cleanly.
* **Pacing Countdown Timer:** Integrates a functional timer to maintain strict testing pacing.
* **Immediate Question Validation:** Provides instant visual feedback on correct or incorrect answers.
* **Performance Analytics Summary:** Displays a final summary score showing percentage evaluations upon quiz completion.

## 🛠️ Tech Stack
* HTML5 & CSS3
* Bootstrap 5 (Responsive Layouts)
* Vanilla JavaScript Engine
